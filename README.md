# Schutzstaffel Cleaner v2.1 (Ultra)

Професійна утиліта на Python для глибокого очищення слідів активності в операційній системі Windows. Програма розроблена для забезпечення максимальної приватності шляхом видалення специфічних згадок про запуск програм, відвідані папки та пошукові запити.

## 🚀 Основні можливості

### 1. Браузери (Chromium)
- **Google Chrome**: Видалення записів з історії відвіданих сайтів та пошукових термінів за ключовими словами.
- **Безпека**: Автоматична перевірка, чи закритий браузер перед початком роботи (для запобігання пошкодженню бази даних SQLite).

### 2. Реєстр Windows (Глибоке сканування)
- **MuiCache**: Видалення історії імен та шляхів запущених програм.
- **ShellBags**: Очищення слідів відкритих папок (навіть якщо вони були на знімних носіях).
- **UserAssist & MRU**: Видалення статистики запусків та історії діалогу "Виконати".
- **Бекапи**: Автоматичне створення `.reg` файлів перед видаленням будь-яких параметрів.
- **Blacklist**: Ігнорування критичних системних гілок (наприклад, `Perflib`) для стабільності ОС.

### 3. Системні журнали та артефакти
- **USN Journal**: Видалення журналу змін файлової системи (через `fsutil`).
- **Prefetch**: Видалення `.pf` файлів, що прискорюють запуск програм.
- **Recent Files**: Очищення ярликів нещодавно відкритих документів.
- **PowerShell History**: Хірургічне видалення рядків з історії команд терміналу.

### 4. Файлова система
- **LocalAppData**: Повне видалення всіх `.cfg` файлів безпосередньо в корені папки `Local` (без втручання в підпапки додатків).
- **Назавжди**: Всі видалення проводяться повз Кошик (Recycle Bin).

### 5. Stealth Mode (Самоочищення)
- Видалення всіх створених бекапів (.reg).
- Видалення згадок про запуск самої утиліти `Schutzstaffel Cleaner` з реєстру та Prefetch.

## 🛠 Технічні вимоги
- **ОС**: Windows 10/11
- **Мова**: Python 3.x
- **Права**: Потребує прав Адміністратора (UAC)

## 📦 Збірка в один EXE файл

Для створення виконуваного файлу використовуйте PyInstaller:
```powershell
pip install pyinstaller
pyinstaller --onefile --uac-admin --name "Schutzstaffel Cleaner" --icon="cc.ico" main.py
```

## 📖 Як користуватися

1. Закрийте браузер Google Chrome.
2. Запустіть зібраний `.exe` файл (він автоматично запитає права адміністратора).
3. Введіть ключові слова через кому (наприклад: `cheat, injector, private`).
4. Виберіть диск для очищення USN Journal (зазвичай `C:`).
5. Після завершення підтвердьте запит на **Самоочищення** (Self-clean), щоб не залишити слідів роботи самої програми.
